2. About

ET is a simple build tool like make, but for literate programming environments.

It allows you to run tasks defined in code blocks in org-mode files, such as this README.org.

Let's define a task: 

#+name: demo1
#+begin_src sh :task yes
  echo "Hey there literates!"
#+end_src

And now let's run it: 

java -jar target/et.jar README.org demo1

~ Task definition {:path "README.org", :args [], :task {}, :task-name "demo1"}
~ Running task demo1 {:task yes}

Hey there literates!

2.1. Command-line arguments

Let's try another example, this time in Ruby: 

#+name: demo2
#+begin_src ruby :task yes
  ARGV.each { |name| puts "Hey there #{name}!" }
#+end_src

Let's run it, passing it random arguments: 

java -jar target/et.jar README.org demo2 Petr Pavel Jakub

~ Task definition {:path "README.org", :args ["Petr" "Pavel" "Jakub"], :task {}, :task-name "demo2"}
~ Running task demo2 {:task yes, :shebang "#!/usr/bin/env ruby"}

Hey there Petr!
Hey there Pavel!
Hey there Jakub!

2.2. Custom shebang

Let's say our code is in Clojure, but we really want to use babashka to run it: 

#+name: demo3
#+begin_src clojure :task yes :shebang "#!/usr/bin/env bb"
  (println "Finally some parenthesis!")
#+end_src

java -jar target/et.jar README.org demo3

~ Task definition {:path "README.org", :args [], :task {}, :task-name "demo3"}
~ Running task demo3 {:task yes, :shebang "#!/usr/bin/env bb", :exports none}

Finally some parenthesis!

2.2.1. TODO Consider supporting header arguments

At the moment ET doesn't really parse Org Mode intelligently, it only scans for the code blocks. Hence it doesn't support:

  • Global header args using #+PROPERTY: header-args :shebang "...".
  • Per-section header args using :PROPERTIES: such as with :header-args: :shebang "...".

Org Mode manual on header arguments

2.3. Documentation and the -T argument

Instead of using :task yes we can use :doc "Task help" 

#+name: demo4
#+begin_src sh :doc "Fourth demo"
  echo "This is the fourth demo"
#+end_src

#+name: demo5
#+begin_src sh :doc "Fifth demo"
  echo "This is the fifth demo"
#+end_src

java -jar target/et.jar -T README.org

~ Task definition {:task {:name :tasks, :fn #object[jakub_stastny.et.runner$fn__190 0x61edc883 "jakub_stastny.et.runner$fn__190@61edc883"]}, :args ("README.org")}
